## Настройка

### 1. Яндекс Карты API ключ (опционально)

Карты работают без ключа, но для production лучше получить свой:

1. Получите бесплатный ключ на https://developer.tech.yandex.ru/
2. Откройте `index.html`
3. Найдите строку: `<script src="https://api-maps.yandex.ru/2.1/?apikey=&lang=ru_RU"`
4. Вставьте ключ после `apikey=`

### 2. Номер WhatsApp ведущего

1. Откройте `index.html`
2. Найдите: `href="https://wa.me/79508611483"`
3. При необходимости замените `79508611483` на актуальный номер ведущего (формат: 79XXXXXXXXX)

### 3. Форма RSVP

Сейчас форма только отображается. Для отправки данных нужно подключить:

- **EmailJS** (бесплатно до 200 писем/месяц) - https://www.emailjs.com/
- **Formspree** (простой сервис) - https://formspree.io/
- Или свой backend
